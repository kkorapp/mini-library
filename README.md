# Mini Library Project

This project was created as part of the PGO course.

## Description
The application simulates a simple library system where books can be created, borrowed, and returned.

## Classes

### Book
Represents a book with:
- title
- author
- page count
- availability status

Methods:
- printInfo()
- borrow()
- returnBook()

### Main
Creates several book objects and demonstrates their behavior.

## Tasks

- Task 1: Created Book class and basic functionality
- Task 2: Added Reader class and methods
- Task 3: Implemented Library class with book storage
- Task 4: Implemented borrowing and returning logic with test scenario
